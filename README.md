# pymath
Extension to the default math module
``` python
import pymath
```
## Install
### Method 1
``` bash
git clone https://github.com/JesterOrNot/pymath.git
python3 setup.py install
```
### Method 2
``` bash
git clone https://github.com/JesterOrNot/pymath.git
bash setup.sh
```
